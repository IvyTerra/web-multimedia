* Multimedia Exercise

A demonstration exercise showing the use of HTML multimedia elements, including audio and video.

* Contents
- HTML file
- CSS file
- Audio file
- Video file (Pixabay source credited in HTML comments)
- Video file from the W3C edX course
- VTT file for subtitles
- Custom video cover image

* Description
This exercise demonstrates how to embed and control multimedia content in HTML, including subtitles and custom video covers.
